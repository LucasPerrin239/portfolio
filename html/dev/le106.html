Dans le cadre d'un projet de fin d'année, nous avions pour objectif de créer un site internet de billetterie. Nous devions créer la base de données, mais également différentes pages, telles que la page de connexion, la page d'inscription nécessitant une validation par mail, et la page de réservation pour un événement, ainsi qu'une partie pour les administrateurs.

Pour ma part, j'ai décidé de réaliser le site du 106, une salle de concert sur les quais de Rouen rive gauche. La demande ne concernait pas le design, mais la fonctionnalité du site. Le but était de manipuler les bases de données grâce au langage MySQL et les requêtes back-end à l'aide du PHP.
